Explorando Tags Básicas

1-Tag h1

É uma tag utilizada para escrever um texto, bem como definir o tamanho da fonte, sendo h1 o maior e h6 o menor

Exemplo:

texto em h1

texto em h2

texto em h3

texto em h4

texto em h5
texto em h6

2-Tag p

É uma tag utilizada para escrever um parágrafo, porém diferentemente da tag citada acima, seu tamanho é definido no CSS. Além disso, esta tag cria um espaço de uma linha na tela, ou seja, o que vier depois do parágrafo vai automaticamente para baixo.

Exemplo:

Este é o primeiro parágrafo

Este é o segundo parágrafo
